What is Yachid?

Yachid is an open source social networking engine. It lets you create public or private social networks that can scale infinitely.

How is Yachid different than other social networking scripts?

We know that there are many social networking scripts out there, some open source, some closed. Some you need to purchase, some you can get for free.

Yachid is different in the sense that: 1) It is the only engine that can scale infinitely. 2) It is the easiest one 3) IT is

Yachid is built on our 10 years experience. we have had social networks that reached millions of members and we know the bottlenecks. We created Yachid

How can Yachid scale indefinitely?

By the power of Amazon Web Services. Yachid is designed from ground up to work with Amazon Web Services. Therefore the only thing you need to do is to enter your Amazon credentials.

Can I run Yachid on my own servers?

Of course you can.

Can I use Yachid for my commercial projects?

Yes you can. You can also take the source code and modify it, as long as you share the modifications with the community. For more details, see Affero Public License.

What's in the name?

Yachid means "one" in Hebrew. Yachid makes groups one.
